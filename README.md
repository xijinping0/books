## 添加新书

| 文件 | 说明 |
| --- | --- |
| `pages/<book>` | 新书内容 |
| `public/<book>` | 新书图片 |
| `_meta.json` | 注册新书目录 |
| `pages/index.md` | 添加新书链接 |
| `theme.config.tsx` | 注册书名 |

## 本地开发

```sh
yarn install
yarn dev
```
